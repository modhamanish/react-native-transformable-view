# react-native-transformable-view

A **React Native Transformable View** component that enables intuitive drag, rotate, and scale interactions on any view. Built with `react-native-reanimated` and `react-native-gesture-handler`, it provides smooth, native‑like performance for pinch‑to‑zoom, rotation gestures, and dragging.

## Features
- Drag, pinch‑to‑zoom, and rotate gestures
- Configurable bounds and aspect‑ratio locking
- Easy to integrate with existing components
- Works on both iOS and Android

## Installation
```bash
npm install react-native-transformable-view
```

## Usage
```tsx
import TransformableView from 'react-native-transformable-view';

<TransformableView>
  <YourContent />
</TransformableView>
```

For more details, see the documentation and examples in the repository.
